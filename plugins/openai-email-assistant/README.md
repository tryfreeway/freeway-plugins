# OpenAI Email Assistant

Draft polished email text from voice input.

## Usage
- Say: "Draft email ..."
- The plugin generates a subject and body.

## Settings
- **OpenAI API key**: Optional. If not set or if the API call fails, the plugin falls back to rule-based formatting.
- **Model**: `gpt-5-nano` or `gpt-5-mini`.
- **Preferred tone**: professional, casual, or concise.

## Notes
- No email is sent automatically.
